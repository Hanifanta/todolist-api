## TodoList - API (Go-Lang)

### Description
This is a simple API for a TodoList. It was developed using Go-Lang and the Golang Fiber framework.

### Features
Activities:
- Create a new activity
- Get all activities
- Get an activity by ID
- Update an activity
- Delete an activity

Todos:
- Create a new todo
- Get all todos
- Get a todo by ID
- Update a todo
- Delete a todo

### Technologies
- Go-Lang
- Golang Fiber
- Gorm
- MySQL
- Docker
- Makefile
